# Triangle Wave MIDI Player

## [Description]
This is a MIDI player that converts your MIDI (.mid) files to triangle wave sounds and allows you to play them in a web browser.

## [How to Use]
1. **Select a File**: Click "Choose File."
2. **Load MIDI**: Select your MIDI (.mid) file.
3. **Play**: Press "PLAY/RESUME" to start playback.

## [Main Features]
* **Play / Pause**: Use **PLAY** to start and **PAUSE** to temporarily stop the playback.
* **Volume Control**: Adjust the **Volume** slider or use the +/- buttons to set your preferred level.
* **Pitch Shift**: Change the key in semitone increments using the **Pitch** control.
* **Precision Tempo**: Fine-tune the speed with 0.01x accuracy using the **Tempo** slider and buttons.
* **Audio Mode**: Toggle between **STEREO** and **MONO** for different spatial experiences.
* **Loop Playback**: Turn **LOOP** ON/OFF for continuous, repeated playback.
* **Waveform Selection**: Switch between **Triangle, Sine, Square, and Sawtooth** waves to change the sound character.
